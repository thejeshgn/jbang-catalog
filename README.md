# My jbang-catalog

## Add the catalog
Add the catalog by running
```bash
jbang --verbose catalog add thejeshgn https://raw.githubusercontent.com/thejeshgn/jbang-catalog/main/thejeshgn-catalog.json
```



## Run a script
For example run tabula cli

```bash
jbang tabula@thejeshgn --help
```
